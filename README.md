## Welcome to blockvhain-engine

This project is an open-source blockchain engine based on Golang.

## Getting Started

Clone the project:

```
git clone git@github.com:vgsantoniazzi/blockchain-engine.git
```

## Usage

##### Execute

```
make app.server
```

##### Compile

```
make
```

##### Run

```
bin/blockchain
```

##### Format code

```
make format
```

##### Clean

```
make clean
```
